# Desafio de Modelagem e Implementação do Componente iPhone

Este desafio foi realizado durante o bootcamp da DIO (Digital Innovation One) focado na Cloud AWS e consiste em uma atividade prática de modelagem e implementação de um componente representando um iPhone.

## Objetivo
Com base no vídeo de lançamento do [1° iPhone 2007](https://www.youtube.com/watch?v=9ou608QQRq8), o objetivo deste desafio é criar uma representação em UML das classes e interfaces que descrevem as funcionalidades do iPhone. Em seguida, você deve implementar essas classes e interfaces em arquivos Java.

## Funcionalidades Representadas
O componente iPhone terá as seguintes funcionalidades representadas:

### Reprodutor Musical
- Tocar música
- Pausar música
- Selecionar música

### Aparelho Telefônico
- Realizar ligações
- Atender chamadas
- Iniciar correio de voz

### Navegador na Internet
- Exibir páginas da web
- Adicionar novas abas de navegação
- Atualizar páginas web
